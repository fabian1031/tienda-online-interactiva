## Descripción del proyecto

Landing page interactiva para una tienda online de paquetes de viajes. Permite explorar destinos turísticos, agregar productos al carrito de compras, eliminar productos y actualizar cantidades. El carrito persiste usando localStorage, por lo que los productos seleccionados se mantienen aunque el usuario recargue o cierre la página.

## Categoría elegida

✈️ Viajes / Paquetes Turísticos

## Tecnologías utilizadas

* HTML5 semántico
* CSS3 (con variables y diseño responsivo)
* Bootstrap 5.3
* JavaScript (localStorage, manipulación del DOM)

## Funcionalidades

* Catálogo de productos renderizado dinámicamente desde un array de objetos
* Agregar productos al carrito
* Eliminar productos del carrito
* Actualizar cantidades (+ / -)
* Persistencia del carrito con localStorage
* Diseño responsive (móvil, tablet, escritorio)
* Animaciones y transiciones CSS
